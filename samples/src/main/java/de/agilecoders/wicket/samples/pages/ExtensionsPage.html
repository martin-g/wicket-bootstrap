Jasny FileUpload

                            


                            
A better looking File Upload input field by Jasny Bootstrap extensions. See issue #130.


                            

                                

                                    

                                        
 
Select fileChangeRemove
                                    

                                

                                

                                    In your java code just use de.agilecoders.wicket.javascript.jasny.FileUploadField instead of Wicket's org.apache.wicket.markup.html.form.upload.FileUploadField.
For HTML markup use any of the recommended snippets at Jasny FileUpload.
